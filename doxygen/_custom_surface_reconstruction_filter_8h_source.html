15 // .NAME CustomSurfaceReconstructionFilter - reconstructs a surface from unorganized points
16 // .SECTION Description
17 // CustomSurfaceReconstructionFilter takes a list of points assumed to lie on
18 // the surface of a solid 3D object. A signed measure of the distance to the
19 // surface is computed and sampled on a regular grid. The grid can then be
20 // contoured at zero to extract the surface. The default values for
21 // neighborhood size and sample spacing should give reasonable results for
22 // most uses but can be set if desired. This procedure is based on the PhD
23 // work of Hugues Hoppe: http://www.research.microsoft.com/~hoppe
24
